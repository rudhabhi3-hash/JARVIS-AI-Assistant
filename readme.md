# JARVIS AI Assistant

A Python-based AI personal assistant inspired by JARVIS.

## Features
- Voice command recognition
- Text-to-speech response
- Wikipedia search
- Opens websites like Google, YouTube, GitHub
- Tells current time and date

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the program:
   python jarvis.py
